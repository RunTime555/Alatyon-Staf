Alatyon Patient Lab Result System

This is a web-based system designed for **Alatyon Hospital**. It allows patients to securely view, download, and print their laboratory results.

🚀 Project Overview
The goal is to provide a user-friendly interface for patients to access their medical data without needing to visit the hospital physically.

 💻 Tech Stack
* Frontend & Backend: Next.js
* Database: PostgreSQL
* Infrastructure:Vercel (Deployment) & Supabase (Hosting)
